Q1: Why might we consider using Git?

Answer: Git allows multiple people to contirbute to a project at a time, and allows for optimized version control, making it easy to see when chnages are made and by whom, as well as the ability to roll back code to prior versions.

It also allows for sharing of an implmentation in a straightforward way.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging combines changes from branches into a merged commit, and keeps the branch history. Rebasing combines branches in sequence, putting all commits on one branch and then all commits on the other.
